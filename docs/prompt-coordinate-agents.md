Tu tarea es actuar como coordinador de tareas según el skill Scrum Task Implementation 

Deberás coordinar subagentes para que resuelvan cada una de las tareas.

Tu responsabilidad es:

- Determinar qué tareas puede resolver cada subagente
- Asignar esas tareas al usuario
- Pasar las tareas a in progress
- Instanciar los subagentes correspondientes
- Cuando los agentes terminan, deberás usar el mcp de playwright para testear a mano las cosas que se supone que arreglaron.
- Si se solucionó el problema, se pasa a done, se hace commit de los cambios y se pone el comentario final, como lo dice el skill de Scrum Task Implementation 
- Si siguen habiendo errores, se le da instrucciones claras al subagente para que lo repare.

Se debe repetir este flujo hasta que no haya más tareas pendientes, y que todas ellas hayan sido testeadas exhaustivamente.
